The Iris Dataset

The Iris flower dataset is set of measurement, specifically speal width and lenght, taken in 1936 by the British statistician and biologist Ronald Fisher and used for the paper The use of multiple measurements in taxonomic problems to illustrate the statistical technique of linear discriminant analysis.

It refers to the measurements taken on several specimens of three types of flowers of the Iris family (see figure)

Iris virginica Iris setosa Iris versicolor

The dataset now is used by basically all the introductory textbooks on Machine Learning as a basic exmple. See for example this tutorial of a famous Python Library on https://scikit-learn.org/stable/auto_examples/datasets/plot_iris_dataset.html".
